Timer Configuration(타이머 구성)
============
1. TIM1 and TIM8

    RCC Clock : APB2 84 MHz

    Only 16-bit interface

2. TIM2 to TIM5

    RCC Clock : APB1 42 MHz

    32- : TIM2, TIM5
    
    16- : TIM3, TIM4

3. TIM9 to TIM11

    RCC Clock : APB2 84 MHz

    Only 16-bit interface

4. TIM12 to TIM14

    RCC Clock : APB1 42 MHz

    Only 16-bit interface
